# Sample anonymised patient data

This folder contains sample anonymised patient data for testing purposes only. No real patient information is present.

Numbers like NIK IDs and names that look realistic are not real:

- **Patient names** are replaced with made-up names that are consistent within the same dataset (so rows still line up for analysis)
- **National ID numbers (NIK)** are replaced with short numeric codes that don't resemble the originals
- **Phone numbers and insurance numbers** are replaced with similar coded substitutes
- **Medical record numbers** are removed — those cells are left blank
- **A set of sensitive columns** — including address, occupation, parents' names, staff names, symptoms, duration of illness, treatment, education given, nursing actions, notes, triage, pharmacy, and registration/referral details — have all their values deleted; column headers are kept so the table structure is preserved
- **Facility names** are replaced with neutral placeholder names everywhere they appear: in cell values, sheet names, and filenames
